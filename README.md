# mxxws 的个人网站

这是 **mxxws** 的个人博客网站，基于 [Astro](https://astro.build) 框架构建。

## 🚀 本地开发

测试11031

在项目根目录下运行以下命令即可启动本地开发服务器：

```sh
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

打开浏览器并访问 `http://localhost:4321` 即可预览网站。

## 🌐 网站部署

本项目已配置 GitHub Actions 工作流。将代码推送到 GitHub 仓库的 `main` 分支后，会自动触发构建并部署到 GitHub Pages。

## 📝 撰写文章

你可以在 `src/content/blog/` 目录下添加或修改 Markdown (`.md`) 或 MDX (`.mdx`) 文件来发布新的博客文章。
